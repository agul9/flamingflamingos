# The Flaming Flamingos
A web-based JavaScript game where players dodge falling fireballs, collect fish, and manage lives.

**[Play here!](https://agul9.github.io/flamingflamingos/)**

### My Role
Responsible for core game development, including player movement, collision detection, and life-state management.

### Tech Stack
JavaScript, HTML/CSS

## How to Play
Help keep our flamingo safe by avoiding the fireballs!
1. Use the arrow keys to move left and right
2. Collect the fish to gain a life (can have up to three lives at a time)
3. Avoid the fireballs or lose a life
4. If you run out of lives, you lose!
